# Blog de Notas 

Este proyecto se trata de un blog de notas basico usando Java como principal lenguaje.

## Descripción

BlogDeNotasJava es una aplicación de escritorio tipo bloc de notas que demuestra el uso de:

- Programación Orientada a Objetos (POO)
- Interfaz gráfica con Swing
- Manejo de eventos
- Lectura y escritura de archivos en Java

## Funciones
- Crear archivo.
- Guardar archivo.
- Editar archivo.

## Herramientas
- Java usando neatbeans.
- Java Swing para interfaz grafca.

## Proposito

Practica en el lenguaje java y swing para interfaces graficas.
